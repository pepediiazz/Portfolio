Este proyecto es una aplicación web llamada YouLirics, diseñada para extraer las letras de canciones a partir de enlaces de YouTube. Fue desarrollada utilizando Angular, Node.js, HTML, CSS y TypeScript. La aplicación utiliza tecnologías avanzadas como Whisper para transcribir audio, Spleeter para separar voz e instrumentos, y FFmpeg para mejorar la calidad del audio. Emplea contenedores Docker para manejar el procesamiento intensivo de datos y fue diseñada para ser escalable y eficiente.
